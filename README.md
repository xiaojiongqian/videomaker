# VideoMaker - 视频处理工具集

基于 Claude Code 的视频处理工具，支持自动字幕生成和智能视频剪辑。

## 功能

- **字幕生成** - 从视频文件中提取音频，自动识别语音并生成字幕
- **智能剪辑** - 分析视频内容，自动移除沉默、重复、口误等冗余片段
- **视频转文档** - AI 分析视频内容，生成配图的结构化总结文档

## 命令列表

### /subtitle - 生成字幕
为视频文件自动生成 SRT 格式字幕。

### /smart-cut - 智能剪辑
分析视频内容，自动识别并剪除冗余片段。

### /video2doc - 视频转文档
从视频生成配图的结构化总结文档。

## 依赖

- FFmpeg
- OpenAI Whisper
